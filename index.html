Método O que faz
.split() Separa uma string em vários itens de array, com base no que foi colocado dentro dos parêntesis. Dada a variável let texto = "Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolor aperiam facere voluptate", ao aplicar o método splic da seguinte maneira: texto.split(" "), retorna o array (12) ["Lorem", "ipsum", "dolor", "sit", "amet", "consectetur", "adipisicing", "elit.", "Dolor", "aperiam", "facere", "voluptate"]. Ao aplicar usando texto.split("e"), retorna a string (10) ["Lor", "m ipsum dolor sit am", "t cons", "ct", "tur adipisicing ", "lit. Dolor ap", "riam fac", "r", " voluptat", ""] e assim por diante.
.slice() O método slice pega uma parte da string, essa parte é definida dentro dos parêntesis. texto.slice(2) retorna a string com o texto a partir do segundo caractere, texto.slice(8) retorna o texto a partir do oitavo caractere. Números negativos retornam os últimos caracteres da string. Ao escolher 2 números separados por vírgula, o texto.slice(a, b) retora a partir a partir do caractere de posição a, até o caractere de posição b.
.substring() Faz basicamente a mesma coisa que o .slice(), se este apresentar 2 valores entre os pareêntesis, porém não é possível usar valores negativos, tendo em vista que o javascript automaticamente troca os números de lugares se o primeiro valor for maior que o segundo.
.substr Não tem no canvas mas eu achei mais legal que o substring então vou falar dele também. É praticamente igual ao substring, mas o segundo valor dentro dos parêntesis equivale ao número de caracteres a partir do primeiro, então (2, 4) pega 4 caracteres a partir do segundo, no substring pegaria do caractere de posição 2 até o caractere de posição 4.
.indexOf() O método indexOf de alguma string qualquer retorna a primeira posição encontrada de uma letra, palavra ou qualquer trecho de texto que esteja presente na string. "O tempo voa".indexOf("o") retorna 6, da palavra tempo. Como deve ter percebido, indexOf é case sensitive. Uma maneira interessante de checar se um trecho de texto está ou não em uma string é usando o if ("texto".indexOf("o") !== -1)(espera-se retorno diferente de -1, então a afirmação é true), porque quando o trecho não está no texto o retorno é -1.
.lastIndexOif() Semelhante ao .indexOf(), porém a string começa a ser contada pelo último caractere, apesar do index ser contado como a palavra foi escrita, ou seja, conta de trás pra frente mas o index é de frente pra frente mesmo.
.includes() Esse método verifica se a string dentro dos parêntesis estão dentro de outra string, se a posição da palavra não importa, esse até agora é o método mais indicado para procura de trechos de texto em outro texto.
.startsWith() Verifica se uma string começa com a string especificada dentro dos parêntesis, é literalmente o que o nome diz. Assim como os outros métodos acima, ao colocar um número depois da string, a busca iniciará a pesquisa a partir do indíce equivalente ao número. Por exemplo "texto".startsWith("ex", 1) retorna true, o trecho ex começa no índice 1 da string "texto".
.endsWith() Semelhante ao método acima, porém, começa contando do fim. Geralmente a primeira letra do trecho é o index, nesse caso a última letra do trecho é o index, que é contado normalmente sem inverter a string.
.trim(), .trimRight, e .trimLeft Remove os espaços do começo e/ou do fim da string. Mas só esses. todos os espaços que estiverem depois da primeira caracteres válido ou antes do último continuarão. .trim remove dos 2 lados, se tiver dos 2 lados, .trimRight remove da direita e .trimLeft remove da esquerda. Existem funções melhores pra remover espaços, mas caso seja necessário esse nível de especificidade, tá aí.
.toUpperCase e .toLowerCase Colocam o texto todo da string em letras maiúsculas e minúsculas, respectivamente.
.match() e .search() Envolve expressão regular, quero ver isso ainda não mó dor de cabeça, se tá nas opcionais(a partir do .trim tava opcional no canvas) então tem jeito melhor de fazer seja lá o que essa tal match e search fizerem.
.concat Concatena 2 arrays de strings ou números, coloca tudo numa array só e retorna outro array. O tal do [...[array]] faz isso melhor.

Agora são os métodos de arrays.

.join() Separa os itens de array com o que tiver dentro dos parêntesis, se não tiver nada neles separa com vírgula sem espaço, uma string vazia só junta tudo. O retorno desse método é uma string.
.splice() Método de array que até onde entendi remove alguns itens do array e adciona outros, opcionalmente. Dentro dos parêntesis vão 2 números e outro valor, que pode ser uma string, uma função e o que mais puder ser colocado como item de array.
.slice() Esse slice é igual o outro slice, só que ao invés de retornar um caractere de um índice, ele retorna um array com os itens dentro dele correspondentes. o array [1, 2, 3, 4].slice(1, 2) retorna tudo o que vier a partir do índice 1 até o índice 2, no caso um único item do array, que é o 2.
.includes() É o includes do string, só que com array. Faz a mesma coisa só que com itens de arrays ao invés de fazer com caracteres. Retorna true se tiver e false se não tiver, tudo igual. Igualmente o método de string, esse conta com 2 parâmetros, o segundo opcional, que se negativo conta a partir do último item e retorna a partir dele em direção ao último. É meio confuso essa parte.
.findIndex() Encontra o index de um item de array, dentro dos parêntesis vai uma função callback, aquelas que define na hora e morre ali mesmo, mas também dá pra colocar funções de fora. Prefiro a primeira opção. Nesse caso, não sei se é padrão, mas a função callback é definida do seguinte modo: [2 ,3 ,4 ,5 ,67 ,1].findIndex(element => element === 3). Nesse caso procura o index de um item da função se, este for igual a 3.
.find Faz a mesma coisa que o método acima, porém retorna o item do array ao invés de retornar o index dele.
.pop() Retorna o último item de um array. Só isso. Pode ser armazenado numa variável. Nada vai entre os parêntesis. Reduz o tamanho da array usada.
.push() Adciona o que tiver entre os parêntesis ao array, no fim dele para ser mais específico.
.shift() Retorna o primeiro item de um array. Só isso. Pode ser armazenado numa variável. Nada vai entre os parêntesis. Reduz o tamanho da array usada.
.unshift() É o push só que no começo do array.
.concat() É literalmente o mesmo que tava na lista dos métodos de string, pq ele atua em string e arrays.
